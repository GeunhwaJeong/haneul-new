## Description

Online IDE for Haneul smart contract development

## Features
- Move
    - Move 2024 is supported
    - Compilation
    - Unit Testing
    - Deployment
    - Only support one default `haneul` binary version
- Project Management
    - Multiple workspaces
    - Persistent session
    - Import from local file system
- Utilities
    - Lightweight object explorer
    - Lightweight package explorer
    - Package function call
- Example templates
