## Description

Haneul Explorer for your localnet.

Haneul Explorer Local is integrated into [Haneul dApp Starter](https://github.com/haneulware/haneul-dapp-starter?tab=readme-ov-file) and [Haneulbase](https://github.com/chainmovers/haneulbase).

## Features
- Object and transaction data view
- Supported networks:
    - Local (default)
    - Custom nodes
