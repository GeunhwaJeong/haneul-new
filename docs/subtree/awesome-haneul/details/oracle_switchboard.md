## Features
- [On-demand data feed](https://docs.switchboard.xyz/docs) (Haneul is not supported)
- [Push-based data feed](https://docs.switchboard.xyz/docs/switchboard/switchboard-v2-push) (Haneul is supported):
    - [Haneul SDK](https://github.com/switchboard-xyz/haneul-sdk)
    - [Haneul examples](https://github.com/switchboard-xyz/haneul-sdk/tree/main/programs/mainnet/feed-parser/sources)
    - [Haneul supported feeds](https://app.switchboard.xyz/haneul/mainnet)
- Build custom feed through [portal](https://app.switchboard.xyz/build)
- Generic data feed protocol allowing devs to build their own feed with customizable oracle jobs. Similar to ChainLink
