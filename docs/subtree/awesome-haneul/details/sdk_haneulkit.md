## Description

HaneulKit is a Swift SDK natively designed to make developing for the Haneul Blockchain easy.

## Features

- [Features](https://github.com/OpenDive/HaneulKit/tree/main?tab=readme-ov-file#features)
- ⚠️ `Bech32` encoded private key is not supported.
- ⚠️ GraphQL is partially supported.
- [Haneul BCS types are supported](https://github.com/OpenDive/HaneulKit/tree/main/Sources/HaneulKit/Types)
- ⚠️ [Kiosk is supported](https://github.com/OpenDive/HaneulKit/tree/main/Sources/HaneulKit/Types/Structs/Kiosk) (might not be actively maintained)
- ⚠️ [HaneulNS is supported](https://github.com/OpenDive/HaneulKit/tree/main/Sources/HaneulKit/Types/Structs/HaneulNS) (might not be actively maintained)
