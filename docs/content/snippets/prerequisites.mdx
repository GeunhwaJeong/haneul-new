<details className="nudge-details">

<summary>

Create Haneul account and setup CLI environment

</summary>

```sh
$ haneul client
```

If this is the first time running the `haneul client` CLI tool, it asks you to provide a Haneul full node server URL and a meaningful environment alias. It also generates an address with a random key pair in `haneul.keystore` and a config `client.yaml`.

By default, the `client.yaml` and `haneul.keystore` files are located in `~/.haneul/haneul_config`. For more information, refer to the [Haneul client CLI tutorial](/references/cli/client.mdx).

If this is not your first time running `haneul client`, then you already have a `client.yaml` file in your local environment. If you'd like to create a new address for this tutorial, use the command:

```sh 
$ haneul client new-address ed25519
```

</details>
