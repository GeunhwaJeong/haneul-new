Move is an open source language for writing safe packages to manipulate on-chain objects (sometimes referred to as "smart contracts"). Move is a platform-agnostic language to enable common libraries, tooling, and developer communities across blockchains with vastly different data and execution models. Move is adaptable to meet the needs of the blockchain the code operates on, see [Move on Haneul](concepts/haneul-move-concepts.mdx#differences) to review enahancements made to Move for optimization on the Haneul blockchain.
