Before being able to publish your package to Devnet, you need Devnet HANEUL tokens. To get some, join the [Haneul Discord](https://discord.gg/Haneul), complete the verification steps, enter the `#devnet-faucet` channel and type `!faucet <WALLET ADDRESS>`. For other ways to get HANEUL in your Devnet account, see [Get HANEUL Tokens](/guides/developer/getting-started/get-coins).

Now that you have an account with some Devnet HANEUL, you can deploy your contracts. To publish your package, use the following command in the same terminal or console:

```
haneul client publish --gas-budget <GAS-BUDGET>
```

For the gas budget, use a standard value such as `20000000`.
