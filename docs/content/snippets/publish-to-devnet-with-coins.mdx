Before being able to publish your package to Testnet, you need Testnet HANEUL tokens. To get some, visit the online faucet at https://faucet.haneul.io/. For other ways to get HANEUL in your Testnet account, see [Get HANEUL Tokens](/guides/developer/getting-started/get-coins).

Now that you have an account with some Testnet HANEUL, you can deploy your contracts. To publish your package, use the following command in the same terminal or console:

```
haneul client publish --gas-budget <GAS-BUDGET>
```

For the gas budget, use a standard value such as `20000000`.
