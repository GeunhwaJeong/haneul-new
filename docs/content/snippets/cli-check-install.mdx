## Check Haneul CLI installation

Before you can use the Haneul CLI, you must install it. To check if the CLI exists on your system, open a terminal or console and type the following command:

```sh
$ haneul --version
```

If the terminal or console responds with a version number, you already have the Haneul CLI installed.

If the command is not found, follow the instructions in [Install Haneul](/guides/developer/getting-started/haneul-install.mdx) to get the Haneul CLI on your system.
