Use one of the following commands for [Homebrew](https://brew.sh/) (MacOS, Linux, or Windows Subsystem for Linux) or [Chocolatey](https://chocolatey.org/) (Windows) to install Haneul.

<Tabs groupId="operating-systems">

<TabItem label="Homebrew" value="mac">

```bash
brew install haneul
```

</TabItem>

<TabItem label="Chocolatey" value="win">

```shell
choco install haneul
```

Find more [versions of Haneul for Windows](https://community.chocolatey.org/packages/haneul) on the Chocolatey community website.

</TabItem>

</Tabs>
