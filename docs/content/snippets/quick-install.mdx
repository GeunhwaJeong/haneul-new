Use one of the following commands for [Homebrew](https://brew.sh/) (MacOS, Linux, or Windows Subsystem for Linux), [Chocolatey](https://chocolatey.org/) (Windows), or [haneulup](https://github.com/GeunhwaJeong/haneulup) (a custom installer and version manager for Haneul related tools) to install Haneul.

<Tabs groupId="install-tool">

<TabItem label="Homebrew" value="brew">

```sh
$ brew install haneul
```

</TabItem>

<TabItem label="Chocolatey" value="choco">

```sh
$ choco install haneul
```

Find more [versions of Haneul for Windows](https://community.chocolatey.org/packages/haneul) on the Chocolatey community website.

</TabItem>

<TabItem label="haneulup (experimental)" value="haneulup">

```sh
$ haneulup install haneul@testnet
```

Find more information on installation and a quick start guide in the haneulup repository: [https://github.com/GeunhwaJeong/haneulup](https://github.com/GeunhwaJeong/haneulup).

</TabItem>

</Tabs>
