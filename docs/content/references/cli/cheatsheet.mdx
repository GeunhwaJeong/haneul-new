## Addresses & Aliases

<table class="w-100">
	<thead>
		<tr>
			<td>Command</td>
			<td>Description</td>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td class="w-2/3">`haneul client active-address`</td>
			<td class="w-1/3">Get the active address</td>
		</tr>
		<tr>
			<td class="w-2/3">`haneul client addresses`</td>
			<td class="w-1/3">List the addresses, their aliases, and the active address</td>
		</tr>
		<tr>
			<td class="w-2/3">`haneul client new-address ed25519`</td>
			<td class="w-1/3">Create a new address with ED25519 scheme</td>
		</tr>
		<tr>
			<td class="w-2/3">`haneul client new-address ed25519 MY_ALIAS`</td>
			<td class="w-1/3">Create a new address with ED25519 scheme and alias</td>
		</tr>
		<tr>
			<td class="w-2/3">`haneul client switch --address ADDRESS`</td>
			<td class="w-1/3">Make this the active address (accepts also an alias)</td>
		</tr>
		<tr> 	
			<td class="w-2/3">`haneul keytool convert PRIVATE_KEY`</td>
			<td class="w-1/3">Convert private key in Hex or Base64 to new format (Bech32 encoded 33 byte flag || private key starting with "haneulprivkey")</td>
		</tr>
		<tr>
			<td class="w-2/3">`haneul keytool generate ed25519`</td>
			<td class="w-1/3">Generate a new keypair with ED25519 scheme and save it to file</td>
		</tr>
		<tr> 	
      <td class="w-2/3">`haneul keytool import INPUT KEY_SCHEME`</td>
			<td class="w-1/3">Add a new key to Haneul CLI Keystore using either the input mnemonic phrase or a Bech32 encoded 33-byte flag || privkey starting with "haneulprivkey"</td>
		</tr>
		<tr>
			<td class="w-2/3">`haneul keytool update-alias OLD_ALIAS NEW_ALIAS`</td>
			<td class="w-1/3">Update the alias of an address</td>
		</tr>
	</tbody>
</table>

		</tr>

				`haneul client pay-haneul \`<br/>&nbsp;&nbsp;`--input-coins COIN_ID \`<br/>&nbsp;&nbsp;`--recipients ADDRESS \`<br/>&nbsp;&nbsp;`--amounts 100000000`

		</tr>

			<td class="w-1/3">Transfer HANEUL object to an address and use the same coin for gas</td>
		</tr>
	</tbody>

## Programmable Transaction Blocks (PTBs)

<table class="w-100">
	<thead>
		<tr>
			<td>Command</td>
			<td>Description</td>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td class="w-2/3">`haneul client ptb --move-call p::m::f "<type>" args`</td>
			<td class="w-1/3">Call a Move function from a package and module</td>
		</tr>
		<tr>
			<td class="w-2/3">`haneul client ptb --make-move-vec "<u64>" "[1000,2000]"`</td>
			<td class="w-1/3">Make a Move vector with two elements of type u64</td>
		</tr>
		<tr>
			<td class="w-2/3">
				`haneul client ptb \`<br/>&nbsp;&nbsp;`--split-coins gas "[1000]" \`<br/>&nbsp;&nbsp;`--assign new_coins \`<br/>&nbsp;&nbsp;`--transfer-objects
				"[new_coins]" ADDRESS`
			</td>
			<td class="w-1/3">Split a gas coin and transfer it to address</td>
		</tr>
		<tr>
			<td class="w-2/3">`haneul client ptb --transfer-objects "[object_id]" ADDRESS`</td>
			<td class="w-1/3">
				Transfer an object to an address. Note that you can pass multiple objects in the array
			</td>
		</tr>
		<tr>
			<td class="w-2/3">
				`haneul client ptb \`<br/>&nbsp;&nbsp;`--move-call haneul::tx_context::sender \`<br/>&nbsp;&nbsp;`--assign sender \`<br/>&nbsp;&nbsp;`--publish "." \`<br/>&nbsp;&nbsp;`--assign
				upgrade_cap \`<br/>&nbsp;&nbsp;`--transfer-objects "[upgrade_cap]" sender`
			</td>
			<td class="w-1/3">Publish a Move package, and transfer the upgrade capability to sender</td>
		</tr>
	</tbody>
</table>
