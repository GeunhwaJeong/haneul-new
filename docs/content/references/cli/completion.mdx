The Haneul CLI `completion` command generates shell completion scripts for the `haneul` command. Install the generated script for your shell to complete Haneul commands, subcommands, flags, and values from your command line.

<ImportContent source="cli-check-install" mode="snippet" />

## Commands

```sh
$ haneul completion --help
```

The command outputs the following help text:

```sh
Generate shell completion scripts for CLI

Usage: haneul completion [OPTIONS] --generate <GENERATOR>

Options:
      --generate <GENERATOR>  If provided, outputs the completion file for given shell [possible values: bash, elvish, fish, powershell, zsh]
  -q, --quiet                 Display less output
  -h, --help                  Print help
  -V, --version               Print version
```

## Generate completion scripts

Redirect the command output to a file for your shell:

```sh
$ haneul completion --generate bash > haneul.bash
$ haneul completion --generate zsh > _haneul
$ haneul completion --generate fish > haneul.fish
```

The command also supports `elvish` and `powershell` values for `--generate`.

## Install completions

Install the generated script in a directory loaded by your shell. Restart your shell after installation, or source the completion file directly if your shell supports it.

### Bash

For Bash, install the generated `haneul.bash` file in a Bash completion directory. Use one of the following common locations:

- Linux: `/etc/bash_completion.d/haneul` for all users.
- Linux: `~/.local/share/bash-completion/completions/haneul` for one user.
- macOS with Homebrew: `$(brew --prefix)/etc/bash_completion.d/haneul`.

For example, to install it for one user on Linux:

```sh
$ mkdir -p ~/.local/share/bash-completion/completions
$ cp haneul.bash ~/.local/share/bash-completion/completions/haneul
```

To load the completion script for your current shell session only, run:

```sh
$ source haneul.bash
```

### Zsh

For Zsh, copy the generated `_haneul` file to a directory in your `fpath`, then rebuild the completion cache:

```sh
$ mkdir -p ~/.zsh/completions
$ cp _haneul ~/.zsh/completions/_haneul
```

Add the directory to `fpath` before `compinit` in your `~/.zshrc` if it is not already configured:

```sh
$ fpath=(~/.zsh/completions $fpath)
$ autoload -Uz compinit
$ compinit
```

After you copy or update the completion file, you might need to remove the Zsh completion cache and start a new shell:

```sh
$ rm -f ~/.zcompdump
```

### Fish

For Fish, copy the generated `haneul.fish` file to your Fish completions directory:

```sh
$ mkdir -p ~/.config/fish/completions
$ cp haneul.fish ~/.config/fish/completions/haneul.fish
```

Fish loads completion files from this directory automatically.
