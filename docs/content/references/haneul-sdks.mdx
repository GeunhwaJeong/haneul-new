Haneul provides developer kits that act as wrappers for the Haneul API. The Haneul community broadens the code coverage with its own set of developer kits targeting the Haneul blockchain. 

## Haneul SDKs

<Cards>
<Card title="dApp Kit" href="https://sdk.haneul-labs.com/dapp-kit">
    A web frontend SDK that interacts with the Haneul API. It is available as an NPM package.
</Card>
<Card title="Rust SDK" href="./rust-sdk">
    SDK configuration and examples of using the Haneul API with Rust, using the `haneul-sdk` crate.
</Card>
<Card title="Rust SDK Auto-generated Docs" href="https://haneullabs.github.io/haneul/haneul_sdk/index.html">
    Auto-generated documentation for the `haneul-sdk` crate in the Haneul repository.
</Card>
<Card title="TypeScript SDK" href="https://sdk.haneul-labs.com/typescript">
    TypeScript SDK for integrating Haneul in your TS apps.
</Card>
<Card title="zkSend SDK" href="https://sdk.haneul-labs.com/zksend">
    zkSend SDK to enable you to incorporate Stashed functionality.
</Card>
</Cards>

## Community SDKs

:::info 

While the community projects are expertly developed, their maintenance and community support vary. You might want to research a project's history and support level before committing to using its utilities. 

:::

<Cards>
<Card title="dApp Kit (Vue)" href="https://github.com/HaneulFansCN/haneulue">
    Haneul dApp Kit for the Vue framework.
</Card>
<Card title="Dart SDK" href="https://github.com/mofalabs/haneul">
    A cross-platform Haneul SDK for mobile, web, and desktop.
</Card>
<Card title="Go SDK" href="https://github.com/block-vision/haneul-go-sdk">
    SDK for developing for Haneul using Golang.
</Card>
<Card title="Kotlin SDK" href="https://github.com/mcxross/khaneul">
    Khaneul is a collection of Kotlin Multiplatform JSON-RPC wrapper and crypto utilities for interacting with a Haneul full node.
</Card>
<Card title="Python SDK" href="https://github.com/FrankC01/pyhaneul">
    pyhaneul is a Python client for developing on the Haneul blockchain. 
</Card>
<Card title="Swift SDK" href="https://github.com/opendive/haneulkit">
    HaneulKit is a Swift SDK natively designed for developing on the Haneul blockchain.
</Card>
</Cards>
