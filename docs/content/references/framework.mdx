## Source code

You can find the source code for these Move modules in the [crates/haneul-framework/packages](https://github.com/GeunhwaJeong/haneul/tree/main/crates/haneul-framework/packages) directory in the `haneul` repository on GitHub. As previously mentioned, the comments included in the code provide context for the logic defined.

## Crate documentation

You can review the raw `cargo doc` output of the following documentation in the `haneul` repository. The .md files are located in the `crates/haneul-framework/docs` directory. Online, they are located at https://github.com/GeunhwaJeong/haneul/tree/main/crates/haneul-framework/docs.

## Build documentation locally

The most recent documentation is always available in the `main` branch of the `haneul` repository. You shouldn't need to build the documentation locally, but if the need arises you can:

1. Open a terminal or console to the `haneul/crates/haneul-framework` directory.
1. Run `cargo doc --workspace --exclude "haneul-benchmark" --no-deps`.
1. The docs are built to `crates/haneul-framework/docs` into their respective subdirectories.

:::info

If the `cargo doc` process does not work as expected, try running `cargo clean` before attempting again. 

:::
