The Legacy Rust SDK crate is in the [`crates/haneul-sdk` directory](https://github.com/GeunhwaJeong/haneul/tree/main/crates/haneul-sdk) of the Haneul repository.

This SDK is forward and backwards compatible with the [`haneul-rust-sdk`](https://github.com/GeunhwaJeong/haneul-rust-sdk) and supports JSON RPC. If you do not need JSON RPC, use the [`haneul-rust-sdk`](https://github.com/GeunhwaJeong/haneul-rust-sdk) instead.
