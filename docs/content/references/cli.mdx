Haneul provides a command line interface (CLI) tool to interact with the Haneul network, its features, and the Move programming language. The complete suite of tools is called the Haneul CLI, with commands grouped together by feature. Each group of commands is commonly referred to by its top-level command: Haneul Client CLI, Haneul Keytool CLI, Haneul Move CLI, and Haneul Validator CLI.

<ImportContent source="cli-check-install.mdx" mode="snippet" />

## Update CLI

The recommended way to manage Haneul CLI installations and versions is via `haneulup`: https://github.com/GeunhwaJeongLabs/haneulup.

:::info

The `tracing` feature is important as it adds Move test coverage and debugger support in the Haneul CLI. Unless it is enabled you will not be able to use these two features.

:::

## Haneul CLI commands

There are a number of top-level commands available, but the six most useful to users are the following. Use the `help` flag for the commands that are not documented yet. For example, `haneul validator --help`.

- **[Haneul Client CLI](./cli/client.mdx):** Use the `haneul client` commands to interact with the Haneul network.
- **[Haneul Client PTB CLI](./cli/ptb.mdx):** Use the `haneul client ptb` command to build and execute PTBs.
- **[Haneul Keytool CLI](./cli/keytool.mdx):** Use the `haneul keytool` commands to access cryptography utilities.
- **[Haneul Move CLI](./cli/move.mdx):** Use the `haneul move` commands to work with the Move programming language.
- **[Haneul Replay CLI](./cli/replay.mdx):** Use the `haneul replay` command to replay a transaction and generate transaction traces for the Move Debugger and trace analysis tools.
- **[Haneul Validator CLI](./cli/validator.mdx):** Use the `haneul validator` commands to access tools useful for Haneul validators.
