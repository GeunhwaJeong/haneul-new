Haneul provides a command line interface (CLI) tool to interact with the Haneul network, its features, and the Move programming language. The complete suite of tools is called the Haneul CLI, with commands grouped together by feature. Each group of commands is commonly referred to by its top-level command or utility: Haneul Client CLI, Haneul Keytool CLI, Haneul Move CLI, Haneul Completion CLI, and Haneul Validator CLI.

<ImportContent source="cli-check-install" mode="snippet" />

## Update CLI

The recommended way to manage Haneul CLI installations and versions is through `haneulup`: https://github.com/GeunhwaJeong/haneulup.

:::info

The `tracing` feature is important because it adds Move test coverage and debugger support in the Haneul CLI. Unless you enable it, you cannot use these two features.

:::

## Haneul CLI commands

There are a number of top-level commands available. Use the `help` flag for the commands that are not documented yet. For example, `haneul validator --help`.

- **[Haneul Client CLI](/references/cli/client):** Use the `haneul client` commands to interact with the Haneul network.
- **[Haneul Client PTB CLI](/references/cli/ptb):** Use the `haneul client ptb` command to build and execute PTBs.
- **[Haneul Completion CLI](/references/cli/completion):** Use the `haneul completion` command to generate shell completion scripts.
- **[Haneul External Keys CLI](/references/cli/external-signers):** Use the `haneul external-keys` commands to manage keys on external hardware signers (Ledger, YubiKey).
- **[Haneul Keytool CLI](/references/cli/keytool):** Use the `haneul keytool` commands to access cryptography utilities.
- **[Haneul Move CLI](/references/cli/move):** Use the `haneul move` commands to work with the Move programming language.
- **[Haneul Replay CLI](/references/cli/replay):** Use the `haneul replay` command to replay a transaction and generate transaction traces for the Move Debugger and trace analysis tools.
- **[Haneul Validator CLI](/references/cli/validator):** Use the `haneul validator` commands to access tools useful for Haneul validators.
