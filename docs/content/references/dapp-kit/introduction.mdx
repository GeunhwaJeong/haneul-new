Haneul Wallet Kit is a set of TypeScript libraries that makes it easy to integrate your dApp with users' wallets in the Haneul ecosystem.

### Core Features {#core-features}

- **Automatic State Management:** The Wallet Kit removes the complexity of state management related to wallet connections. You can focus on building your dApp.
- **Supports all Haneul wallets:** No need to configure adapters or manually define wallets you support. All Haneul wallets are automatically supported.
- **Easy to integrate:** Haneul Wallet Kit provides pre-built React Components that you can drop right into your dApp, for easier integration
- **Flexible:** Haneul Wallet Kit can be used with our React Component, or with just the core React functionality with your own UI. If you're building outside of React, you can use Haneul Wallet Kit Core to benefit from the state management.
