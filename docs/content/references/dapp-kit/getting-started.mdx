### Installation {#installation}

To install the [React Haneul Wallet Kit library from npm](https://www.npmjs.com/package/@haneullabs/wallet-kit), along with the Haneul TypeScript SDK:

```bash
npm install @haneullabs/wallet-kit @haneullabs/haneul.js
```

### Configuring the Haneul Wallet Kit Provider {#configure-haneul-wallet-kit-provider}

After you install the `@haneullabs/wallet-kit` package, you need to wrap your application with the `WalletKitProvider`.

```tsx
import { WalletKitProvider } from '@haneullabs/wallet-kit';

function App() {
  return (
    <WalletKitProvider>
      <YourApp />
    <WalletKitProvider />
  );
};
```

### Rendering a Connect Button {#render-connect-button}

After you wrap your app with the `WalletKitProvider`, add a `ConnectButton` component that users can click to connect their wallet to your dApp.

```tsx
import { ConnectButton } from "@haneullabs/wallet-kit";

export function YourApp() {
  return <ConnectButton />;
}
```
