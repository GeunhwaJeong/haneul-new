Haneul is the first internet-scale programmable blockchain platform. That might read like marketing speak, but as you peruse the documentation to understand the technology, you will discover that Haneul addresses many of the problems that hold blockchains back from mass adoption. 

Before you can get started developing on Haneul, you need to understand the code repository and install its binaries.

## Move IDEs and plugins

The recommended IDE for Move development is Visual Studio Code with the [Move extension](https://marketplace.visualstudio.com/items?itemName=haneullabs.move) available in the Visual Studio Marketplace. Follow the Visual Studio Marketplace instructions to install the extension.

See more [IDE options](https://github.com/GeunhwaJeong/awesome-move#ides) in the [Awesome Move](https://github.com/GeunhwaJeong/awesome-move) documentation.

After you install VS Code and the `Move` extension, check out the [Move code examples](https://github.com/GeunhwaJeong/haneul/tree/main/examples/move).
