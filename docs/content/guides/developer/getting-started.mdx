Haneul is the first internet-scale programmable blockchain platform. That might read like marketing speak, but as you peruse the documentation to understand the technology, you will discover that Haneul addresses many of the problems that hold blockchains back from mass adoption. 

Before you can get started developing on Haneul, you need to understand the code repository and install its binaries. Consequently, the first two topics you need are:

- [Haneul Environment Setup](./getting-started/haneul-environment.mdx)
- [Install Haneul](./getting-started/haneul-install.mdx)

## After installing Haneul

After you have Haneul installed on your system, you can begin developing. You can learn about base aspects of the system in Haneul 101, or jump straight into writing and publishing a package with the instruction in Your First dApp. There is also a growing set of code examples you can draw from, written by Haneul and Move experts. 

- [Haneul 101](./haneul-101.mdx)
- [Your First dApp](./first-app.mdx)
- [Examples](./app-examples.mdx)

## Move

Move is the programming language that powers smart contracts on Haneul. Move is highly adaptable, with several blockchains using the language. Haneul adapts the language to meet the unique needs of the Haneul blockchain, so although you might be familiar with Move on other blockchains, there are features unique to its use on Haneul. For more information on how the language operates on Haneul versus its original use case, see [Move](/concepts/haneul-move-concepts.mdx) in the Concepts section.
