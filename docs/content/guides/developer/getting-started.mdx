Haneul is the first internet-scale programmable blockchain platform. That might read like marketing speak, but as you peruse the documentation to understand the technology, you will discover that Haneul addresses many of the problems that hold blockchains back from mass adoption. 

Before you can get started developing on Haneul, you need to understand the code repository and install its binaries. Consequently, the first two topics you need are:

- [Haneul Environment Setup](./getting-started/haneul-environment.mdx)
- [Install Haneul](./getting-started/haneul-install.mdx)

## After installing Haneul

After you have Haneul installed on your system, you can begin to connect with a network and publish smart contracts. 

- [Connect to a Haneul Network](./getting-started/connect.mdx)
- [Connect to a Local Network](./getting-started/local-network.mdx)
- [Get Haneul Address](./getting-started/get-address.mdx)
- [Get HANEUL Coins](./getting-started/get-coins.mdx)
