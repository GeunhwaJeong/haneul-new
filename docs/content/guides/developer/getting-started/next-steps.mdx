If you have followed the Getting Started path, you should be ready to continue your journey on Haneul. This page details how to continue learning, additional resources, and ways to join the developer community. 

<details>

<summary>Verify "Getting Started" essentials</summary>

To make sure you have everything you need, paste the following commands into a console in your local environment:

```sh
$ haneul --version
$ haneul client active-address
$ haneul client active-env
$ haneul client gas
```

If all commands are successful, your console displays output similar to the following:

```sh
haneul [version_number]-[commit_hash]
0x6543...7241
testnet
╭───────────────┬────────────────────┬──────────────────╮
│ gasCoinId     │ geunhwaBalance (GEUNHWA) │ haneulBalance (HANEUL) │
├───────────────┼────────────────────┼──────────────────┤
│ 0x1e86...63b2 │ 5041161576         │ 5.04             │
╰───────────────┴────────────────────┴──────────────────╯
```

If you receive an error, revisit the content relevant to the command that caused the error or lacked output.

| Command | Successful if |
| ------- | -------------- |
| `haneul --version` | You [installed Haneul](/guides/developer/getting-started/haneul-install.mdx) correctly. |
| `haneul client active-address` | You have a [Haneul address](/guides/developer/getting-started/get-address.mdx) that is currently active. |
| `haneul client active-env` | You have set up a [connection to Testnet](/guides/developer/getting-started/configure-haneul-client.mdx) that is currently active. |
| `haneul client objects` | You address owns [HANEUL for testing](/guides/developer/getting-started/get-coins.mdx). |

</details>

## Additional documentation

The rest of the documentation on this site contains important concepts, guides, and examples for you to continue your Haneul journey. There are also resources beyond this documentation that you can reference, as well as a large and growing community whose experience and knowledge you can leverage to get the most out of Haneul. 

### SDK documentation

Haneul Labs, the company that initiated the creation of Haneul, provides SDKs to interact with the network and associated assets. Visit the [SDK docs](https://sdk.haneul-labs.com/typescript) to learn more.

### The Move Book

Move is the language of smart contracts on Haneul. [The Move Book](https://move-book.com/) details the language.
