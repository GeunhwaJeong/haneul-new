In many education systems, 101-level classes are those that teach core competencies to build a foundation for the more advanced topics that are to come. The topics in this section provide the same experience for the Haneul blockchain. They examine the core Haneul development concepts that you use to build apps on Haneul. As you start developing more advanced solutions, knowledge of these topics provides a good base for understanding the more advanced concepts you will employ.

## Creating coins and NFTs on Haneul

Everything on the Haneul blockchain is an object. These topics use code examples to demonstrate how to create these specific types of objects. 

- [Create Coins](./haneul-101/create-coin.mdx)
- [Create NFTs](./haneul-101/create-nft.mdx)

## Building programmable transaction blocks

You can create programmable transaction blocks (PTBs) on Haneul to perform multiple commands in a single transaction. The [Building Programmable Transaction Blocks](./haneul-101/building-ptb.mdx) topic demonstrates how to build a PTB using the Haneul TypeScript SDK.

## Using events

You can emit events from your published packages on the Haneul network. [Using Events](./haneul-101/using-events.mdx) demonstrates how to emit events from your on-chain packages and monitor the activity of other objects emitting events. 

## Shared versus Owned Objects

Objects on Haneul, unlike other blockchains, can be owned as well as shared. You can create transactions that leverage either type or both. [Shared versus Owned Objects](./haneul-101/shared-owned.mdx) examines the differences and what considerations you should account for when deciding how to structure your on-chain app. 

## Access On-Chain Time

Haneul provides a `Clock` module you can use to get network-based time. [Access On-Chain Time](./haneul-101/access-time.mdx) examines the `Clock` module and the behavior of the available methods that affect transaction processing speed and the temporal exactness of the data you receive.
