In many education systems, 101-level classes are those that teach core competencies to build a foundation for the more advanced topics that are to come. The topics in this section provide the same experience for the Haneul blockchain. They examine the core Haneul development concepts that you use to build apps on Haneul. As you start developing more advanced solutions, knowledge of these topics provides a good base for understanding the more advanced concepts you will employ.
