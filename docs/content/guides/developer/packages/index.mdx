A Move package on Haneul includes one or more modules that define the package's interaction with on-chain objects. You develop the logic for those modules in Move, compile them into an object, and publish that package object to a Haneul network. On-chain, anyone can view a package's contents and see how its logic manipulates objects using a Haneul Explorer.
