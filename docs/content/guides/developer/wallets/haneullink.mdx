[HaneulLink](https://www.haneullink.io/) enables securely linking wallet addresses from different blockchains to a Haneul wallet address. Upon creating a HaneulLink, you receive a soulbound NFT that serves as authenticated proof of ownership for your connected wallets.

## Key features

HaneulLink provides the following capabilities:

- **Cross-chain identity verification**: Link wallets from [Ethereum](/concepts/haneul-for-ethereum.mdx), [Solana](/concepts/haneul-for-solana.mdx), and other Haneul addresses to your primary Haneul wallet.
- **Non-transferable proof of ownership**: Receive a [soulbound NFT](/guides/developer/nft/nft-soulbound.mdx) that cryptographically verifies wallet ownership.
- **Enhanced discoverability**: Projects in the [Haneul ecosystem](https://github.com/haneul-foundation/awesome-haneul/tree/main) can discover you based on your activity across all connected wallets.
- **Reward eligibility**: Unlock opportunities for cross-chain rewards, priority features, and [exclusive events](https://luma.com/haneul-network) within the Haneul network.

### Supported networks

HaneulLink currently supports linking wallets cross-chain between Haneul and Ethereum (Haneul-to-Ethereum), and Haneul and Solana (Haneul-to-Solana).

Haneul-to-Haneul linking is currently only supported programmatically with a custom solution for the [Playtron wallet app](https://wallet.playtron.one/). For more information, visit the [HaneulPlay0X1 HaneulLink integration guide](/guides/haneulplay0x1/wallet-integration.mdx#haneullink).

## Creating a HaneulLink

Creating a HaneulLink incurs a small gas fee on the Haneul network to mint the soulbound NFT. Ensure you have some HANEUL tokens in your wallet to complete the connection. 

To create a HaneulLink:

1. Visit [https://www.haneullink.io/](https://www.haneullink.io/)
2. Click **Connect Haneul Wallet**
3. Select the blockchain network you want to link: Ethereum or Solana
4. Follow the prompts to authenticate and connect the wallet you want to link
5. Complete the transaction to mint your HaneulLink NFT

## Managing HaneulLink

Each HaneulLink connection creates a separate HaneulLink NFT in your Haneul wallet.

To view your linked addresses:

1. View your HaneulLink NFTs in your Haneul wallet
2. Check the `network_address` field to see which wallets are linked
3. Review the timestamp to see when each link was created

You can permanently unlink a wallet by burning the corresponding NFT. To unlink an address:

1. Navigate to the NFTs section of your Haneul wallet
2. For the HaneulLink NFT you want to burn, select "Send" or "Transfer"
3. Enter the burn address: `0x0000000000000000000000000000000000000000`

Additionally, there are various third-party tools (such as [HANEUL Manager](https://www.haneulmanager.com/burner)) that can help burn NFTs, and some wallets have a built-in burning feature.

### Querying HaneulLink objects

To retrieve a user's HaneulLink connections, use the following gRPC query:

```typescript
const HANEULLINK_PACKAGE_ID_ETH_SOL = '0xf857fa9df5811e6df2a0240a1029d365db24b5026896776ddd1c3c70803bccd3';
const HANEULLINK_PACKAGE_ID_HANEUL = '0x73f5ab2461c5993408fff21354fa9831d4f4a66cc81382419ec29e3c80c384b5';

const ethSolHaneulLinks = await grpcClient.listOwnedObjects({
  owner: haneulAddress,
  type: `${HANEULLINK_PACKAGE_ID_ETH_SOL}::haneullink::HaneulLink`,
  include: { json: true },
});

const haneulHaneulLinks = await grpcClient.listOwnedObjects({
  owner: haneulAddress,
  type: `${HANEULLINK_PACKAGE_ID_HANEUL}::haneullink::HaneulLink`,
  include: { json: true },
});
```

This returns all HaneulLink NFTs owned by the specified Haneul address.

### HaneulLink NFT structure

Each HaneulLink NFT contains the following fields:

```typescript
{
  "id": "0xcafe",
  "network_address": "0xdecaf",
  "timestamp_ms": "1751541273947"
}
```

The NFT type indicates which blockchain network the wallet is linked to:

```typescript
for (const link of ethSolHaneulLinks.objects) {
  const id = link.json.id as string;                   // "0xcafe"
  const linkedAddress = link.json.network_address as string; // "0xdecaf"
  const timestamp = link.json.timestamp_ms as string;        // "1751541273947"
  const chainType = link.type; // full type string includes the generic param,
                               // e.g. "...::HaneulLink<...::ethereum::Ethereum>"
}

// Haneul-to-Haneul HaneulLink (Mainnet)
0xf857fa9df5811e6df2a0240a1029d365db24b5026896776ddd1c3c70803bccd3::haneullink::HaneulLink<0x73f5ab2461c5993408fff21354fa9831d4f4a66cc81382419ec29e3c80c384b5::haneul::Haneul>

// Haneul-to-Ethereum HaneulLink (Testnet)
0x0025bafa2e6afa511c19bd4e95626c897e798fde629b4782fe061bdc8bd65c8a::haneullink::HaneulLink<0x0025bafa2e6afa511c19bd4e95626c897e798fde629b4782fe061bdc8bd65c8a::ethereum::Ethereum>

// Haneul-to-Solana HaneulLink (Testnet)
0x0025bafa2e6afa511c19bd4e95626c897e798fde629b4782fe061bdc8bd65c8a::haneullink::HaneulLink<0x0025bafa2e6afa511c19bd4e95626c897e798fde629b4782fe061bdc8bd65c8a::solana::Solana>
```

### Verifying cross-chain asset ownership

After retrieving a user's HaneulLink objects, you can verify their ownership of specific assets on the linked blockchain:

1. Extract the `network_address` from the HaneulLink NFT.
2. Use the appropriate blockchain APIs to query assets at that address, such as [Alchemy](https://www.alchemy.com/) for Ethereum, and [Metaplex](https://www.metaplex.com/) for Solana.
3. Verify the presence of the required asset.
4. Attribute benefits to the user through their Haneul wallet.
