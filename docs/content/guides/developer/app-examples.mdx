The ever-growing number of examples in this section showcase packages for the Haneul blockchain. Extract techniques used in these examples to apply to your own Haneul projects as they are written by Haneul and Move experts. 

## Examples

Haneul is dedicated to providing a wide range of examples to guide you in proper programming techniques for the Haneul blockchain. This list will continue to grow, so check back often.

- [Counter](./app-examples/e2e-counter.mdx): An end-to-end example that creates a basic decentralized counter that anyone can increment, but only the object owner can reset it. The example includes Move code to create the package and leverages the Haneul TypeScript SDK to provide a basic frontend.
- [Coin Flip](guides/developer/app-examples/coin-flip.mdx): The Coin Flip app demonstrates on-chain randomness.
- [Bot Prevention](guides/developer/app-examples/recaptcha.mdx): Allow interactions with your smart contract only from humans, not bots.
