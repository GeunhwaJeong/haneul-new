## Haneul CLI

Interact directly with Haneul networks and its features using the Haneul command line interface (CLI). The CLI is divided into separate base commands that target a specific set of features.

<Cards>
<Card title="Haneul Client CLI" href="references/cli/client">
Create a client on a Haneul network to generate addresses, access networks, and more with the Haneul Client CLI.
</Card>
<Card title="Haneul Client PTB CLI" href="references/cli/ptb">
Build, preview, and execute programmable transaction blocks directly from your terminal with the Haneul Client PTB CLI.
</Card>
<Card title="Haneul Move CLI" href="references/cli/move">
Access Haneul Move functions on chain using the Haneul Move CLI.
</Card>
