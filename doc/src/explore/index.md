See examples and tutorials built on Haneul.

## Haneul applications

[Haneul Capys](https://capy.art/) - Haneul Capys demonstrates the capabilities of Haneul with cute, fun capybaras you can collect, breed, and adorn with your favorite items. 

[Satoshi Flip](https://satoshi-flip.haneul.io/) - Satoshi Flip is the first Haneul application with fair and transparent random selection output, using a Haneul epoch-locked Satoshi-Dice variant.
