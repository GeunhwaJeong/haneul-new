---
title: Connect and Communicate with the Haneul Network
---

Now that you have [installed Haneul](../../build/install.md), [started the Haneul network](../../build/cli-client.md), and learned how to [create smark contracts in Move](../../build/move/index.md) and [program Haneul objects](../../build/programming-with-objects/index.md), it's time to let your apps talk to Haneul. The pages in this section provide various options for communicating with Haneul.

* Use the [Haneul CLI client](../../build/cli-client.md) to start and set up the Haneul network.
* Set up your own [local Haneul RPC server and use the Haneul JSON-RPC API](../../build/json-rpc.md) to interact with a local Haneul network.
* Adhere to the [restrictions placed on JSON types](../../build/haneul-json.md) to make them HaneulJSON compatible.
* Interact with the Haneul network via the [Haneul Rust SDK](../../build/rust-sdk.md), a collection of Rust language JSON-RPC wrapper and crypto utilities.
* Sign transactions and interact with the Haneul network using the [Haneul TypeScript SDK](https://github.com/GeunhwaJeong/haneul/tree/main/sdk/typescript) built on the Haneul JSON RPC API.
* Run a [Haneul Fullnode](../../build/fullnode.md) yourself to store the full Haneul blockchain state and history and qualify as a [potential validator](https://haneul.io/resources-haneul/validator-registration-open/).
* Filter and subscribe to a [real-time event stream](../../build/pubsub.md) on your Haneul Fullnode using JSON-RPC notifications via the WebSocket API.
