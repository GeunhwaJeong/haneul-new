---
title: Connect to the Haneul Network
---

The pages in this section provide various options for communicating with Haneul:

* Use the [Haneul JSON-RPC API](json-rpc.md) to interact with a Haneul network.
* Use the [Haneul CLI client](cli-client.md).
* Interact with the Haneul network via the [Haneul Rust SDK](rust-sdk.md), a collection of Rust language JSON-RPC wrapper and crypto utilities.
* Sign transactions and interact with the Haneul network using the [Haneul TypeScript SDK](https://github.com/GeunhwaJeong/haneul/tree/main/sdk/typescript) built on the Haneul JSON RPC API.
* Run a [Haneul Full node](fullnode.md) yourself to store the full Haneul blockchain state and history.
* Filter and subscribe to a [real-time event stream](pubsub.md) on your Haneul Full node using the WebSocket API.

See the [Haneul Reference Documentation](reference.md) for information on our API and HaneulJSON format.
