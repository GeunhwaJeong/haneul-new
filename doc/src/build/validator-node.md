---
title: Run a Haneul Validator node
---

To set up and run a Haneul Validator node, see [Haneul for Node Operators](https://github.com/GeunhwaJeong/haneul/blob/main/nre/haneul_for_node_operators.md).

To successfully connect a validator node to a Haneul network, you must have the minimum required amount of HANEUL to stake. If you do not, the transaction to join the validator quorum fails.

To learn more about Haneul Validators, see [Validators](../learn/architecture/validators.md).
