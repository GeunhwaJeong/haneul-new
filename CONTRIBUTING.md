# Contributors

Thanks for considering making a contribution to the Haneul network or its documentation. The Haneul monorepo houses the code that powers and documents the Haneul blockchain. 

## Contribute to Haneul

See [Haneul Environment Setup](https://github.com/GeunhwaJeong/haneul/blob/main/docs/content/guides/developer/getting-started/haneul-environment.mdx) for approach to submitting code fixes and enhancements.

If you want to contribute code that creates a feature on Haneul, start with a [Haneul Improvement Proposal](https://github.com/haneul-foundation/sips/tree/main) before developing the logic.

Found a bug or security vulnerability? Create a [GitHub issue](https://github.com/GeunhwaJeong/haneul/issues/new/choose). 

Found a small error or typo in the documentation? Each page on the [haneul.docs.io](https://haneul.docs.io) site includes an **Edit this page** link at the bottom that you can use to edit the page in GitHub. The content is located in the docs/content directory of the Haneul repo, so you can make a usual PR if you prefer. 

For larger documentation issues, you can [create an issue](https://github.com/GeunhwaJeong/haneul/issues/new/choose) in GitHub. To fix the problem yourself, follow the [documentation contribution](./docs/content/references/contribute/contribution-process.mdx) guidelines.

Want to contribute to the Haneul framework? See [CONTRIBUTING.md](https://github.com/GeunhwaJeong/haneul/blob/main/crates/haneul-framework/CONTRIBUTING.md) for information related to `haneul-framework` crate contributions.
