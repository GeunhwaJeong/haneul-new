# Connecting to Haneul Network

The Haneul TypeScript SDK provides a HaneulClient class that you can use to connect to a network's
JSON-RPC server. Use the `HaneulClient` for all JSON-RPC operations.

## Network locations

The following table lists the locations for the Haneul network and the default local network settings.

| Network | Full node                             | faucet                              |
| ------- | ------------------------------------- | ----------------------------------- |
| local   | `http://127.0.0.1:9000`               | `http://127.0.0.1:9123/gas`         |
| Devnet  | `https://fullnode.devnet.haneul.io:443`  | `https://faucet.devnet.haneul.io/gas`  |
| Testnet | `https://fullnode.testnet.haneul.io:443` | `https://faucet.testnet.haneul.io/gas` |
| Mainnet | `https://fullnode.mainnet.haneul.io:443` | `null`                              |

## Using HaneulClient to connect to a Haneul network

To establish a connection to a network, import `HaneulClient` from `@haneullabs/haneul.js/client` and pass the
relevant URL to the `url` parameter. The following example establishes a connection to Testnet and
requests HANEUL from that network's faucet.

```typescript
import { HaneulClient } from '@haneullabs/haneul.js/client';
// assign Testnet RPC address
const haneulNetwork = 'https://fullnode.testnet.haneul.io:443/';
// const haneulNetwork = 'https://fullnode.devnet.haneul.io:443'; // Devnet connect
// const haneulNetwork = 'https://fullnode.mainnet.haneul.io:443'; // Mainnet connect
// const haneulNetwork = 'http://127.0.0.1:9000'; // Local network connect (default address)
// set url value to Testnet RPC URL
const haneulClient = new HaneulClient({ url: haneulNetwork });
// get tokens from the Testnet faucet server
await haneulClient.getCoins({
	owner: '<OWNER_ADDRESS>',
});
```

For local development, you can run `cargo run --bin haneul-test-validator` to spin up a local network
with a local validator, a Full node, and a faucet server. Refer to
[the Local Network guide](https://docs.haneul.io/build/haneul-local-network) for more information.
