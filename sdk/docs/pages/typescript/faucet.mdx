# Faucet

import { Callout } from 'nextra/components';

Devnet, Testnet, and local networks include faucets that mint HANEUL. You can use the Haneul TypeScript
SDK to call a network's faucet and provide HANEUL to the address you provide.

To request HANEUL from a faucet, import the `requestHaneulFromFaucetV0` function from the
`@haneullabs/haneul.js/faucet` package to your project.

```typescript
import { getFaucetHost, requestHaneulFromFaucetV0 } from '@haneullabs/haneul.js/faucet';
```

Use `requestHaneulFromFaucetV0` in your TypeScript code to request HANEUL from the network's faucet.

```typescript
await requestHaneulFromFaucetV0({
	host: getFaucetHost('testnet'),
	recipient: <RECIPIENT_ADDRESS>,
});
```

<Callout type="info">
	Faucets on Devnet and Testnet are rate limited. If you run the script too many times, you surpass
	the limit and must wait to successfully run it again.
</Callout>
