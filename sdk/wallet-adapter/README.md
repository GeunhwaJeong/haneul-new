# Haneul Wallet Adapter

> **⚠️ These packages are experimental and will change rapidly as they are being developed. Do not consider these APIs to be stable. If you have any feedback, [open an issue](https://github.com/GeunhwaJeong/haneul/issues/new/choose) or message us on [Discord](https://discord.gg/Haneul).**

Haneul Wallet Adapter is a set of libraries that makes it easy to connect your dApp to Haneul wallets.

## Getting started

We recommend using [Haneul Wallet Kit](./packages/wallet-kit/README.md) for connecting to wallets from React applications.

If you're not using React, you can use the underlying [wallet adapter packages](./packages/adapters/) to determine which wallets are currently available in the dApp, and interact with the wallets.
